Project Title: Bank Management System

📖 Project Description:

This project is a Bank Management System developed using Java Swing for GUI and MySQL for database management. It simulates the basic functionalities of an ATM system where users can perform banking operations securely.
The system allows users to create an account, log in using card number and PIN, and perform various transactions like deposit, withdrawal, balance enquiry, and more.

🚀 Features:
🔐 User Login Authentication (Card Number & PIN)
📝 New Account Registration (Multi-step form)
💰 Deposit Money
💸 Withdraw Money
⚡ Fast Cash (Quick withdrawal options)
📊 Balance Enquiry
🧾 Mini Statement (Transaction history)
🔄 PIN Change

🛠️ Technologies Used:
Java (Swing, AWT) → GUI Design
MySQL → Database
JDBC → Database Connectivity
OOP Concepts → Code Structure

⚙️ How It Works:
1.User signs up → account created in database
2.User logs in using card number & PIN
3.User selects transaction type
4.System updates database accordingly
5.Result is displayed on GUI
